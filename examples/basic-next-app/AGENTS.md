# Project Agent Guide

## Project Overview

This is a Next.js + TypeScript project.

- Framework: Next.js
- Language: TypeScript
- Package manager: npm
- Tests: Vitest
- Linting: ESLint

## Commands

- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`
- Typecheck: `npm run typecheck`
- Test: `npm test`

## Coding Standards

- Use TypeScript strict mode.
- Follow ESLint rules.
- Format with Prettier.

## Testing Rules

- Run tests before committing.
- Use Vitest for unit tests.
- Write tests for all new features.

## Security Rules

- Never commit secrets or API keys.
- Validate user input on server side.
- Keep dependencies updated.

## Known Constraints

- This is a sample project for ContextKit testing.
